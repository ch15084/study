#JVM
##  1   概述
###  1.1 关键词
    jvm: java virtual machine           #java虚拟机
    jre: java run-time environment      #java运行时环境
    jdk: java development kit           #java开发组件
    jit: just-in-time compilation       #即时编译
###  1.2  知识点
    jvm内存模型
    gc:垃圾收集器
    类加载机制
    jvm性能监控及调优
